Candy Crush
HTML CSS JavaScript Node.js Express MongoDB

https://candycrushgame.herokuapp.com/

A Candy Crush game, similar in functionality to the original. Entire game built using HTML, CSS, and JavaScript with no frontend frameworks or graphics library. The project includes several algorithms that determine all possibile candy create and crush combinations. Players have 20 turns to try and get the highest score they possible can and at the end, may register their score, which is stored in MongoDB via Express and Node.js. A leaderboard of player and scores are displayed at the end of the game.
