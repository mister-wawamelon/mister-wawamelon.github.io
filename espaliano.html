Espaliano
Python Django MySQL HTML CSS Materialize jQuery Google Translate API Tesseract OCR

Espaliano is a Django web application designed for Spanish and Italian learners. The application provides lessons, organizes vocabulary words by category, and highlights lexical similarities between the 2 languages. A translation feature is also provided via Google Translate API and jQuery. Users may type in text for translation or upload images of texts, which are extracted using Tesseract OCR.

Currently, vocabulary and lessons content is still being added.
