Py2Ruby
Python Flask Ruby HTML CSS Materialize jQuery JavaScript

https://py2ruby.com/

Py2Ruby is a web application built with Flask that translates Python 3.7+ code to Ruby 2.7+ code. An editor is built into the page to provide a code input environment. Py2Ruby works by using a syntax parser and translator module that I wrote myself in Python. JQuery is used to submit the input Python code for conversion.

Py2Ruby has been tested thoroughly and has accurately and fully translated over 1000+ lines of python code in one setting. Tests have also been conducted successfully on implementations of various searching and sorting algorithms, as well as data structures implementations such as ArrayStack, Linked List, TreeMap, ArrayHeap, UnionFind/DisjointSet.
