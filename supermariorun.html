Super Mario Run
HTML CSS JavaScript Canvas.js Adobe Animate

Super Mario Run is an infinite runner minigame built with Canvas.js, JavaScript, and designed with HTML, CSS. Adobe Animate was used to create a JSON spritesheet. Mario continues to run as platforms are generated indefinitely from the right. Koopas and Goombas are also generated at random and the player can jump to avoid the monsters. Coins also appear and the goal of the game is to collect as many coins without falling off the platform or colliding with the monsters.
