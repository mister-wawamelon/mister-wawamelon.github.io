Pokemon Mystery Dungeon Quiz
Python Flask HTML CSS JavaScript jQuery

https://pokemonmdquiz.herokuapp.com/

Recreation of the starter Pokemon personality quiz from the Pokemon Mystery Dungeon series. The quiz consists of 8 questions randomly chosen from a set of 56. Depending on the user's response, points are added to a list of character traits. The character trait with the most points is selected for the user's nature. The 9th question then asks if the person is male or female, which along with the nature, will be used to determine what Pokemon the user will be.
